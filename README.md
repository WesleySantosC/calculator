# Python Terminal Calculator

This project is a simple calculator that runs in the terminal.  
It supports basic math operations and keeps a history of calculations.

You can save the history to a `.txt` file using the command `save history` and clear it using `clear`.

---

## Cloning the project

To clone this repository, run:

```bash
git clone https://github.com/WesleySantosC/calculator.git
```

---

Accessing the project

Navigate into the project folder:

```bash
cd calculator
```

---

Running the project

Run the program with:

```bash
python3 main.py
```

---

### Available commands
``` 
    + ➜ Addition 
```
``` 
    - ➜ Subtraction
```
``` 
    * ➜ Multiplication 
```
``` 
    / ➜ Division 
```
```
    % ➜ Modulo
``` 
``` 
    ** ➜ Power
```
``` 
    history ➜ Show calculation history
```
``` 
    save history ➜ Save history to a .txt file
```
``` 
    clear ➜ Clear history
```
``` 
    exit ➜ Exit the program
```

---

### Features
* Terminal-based calculator
* Calculation history system
* Save history to file
* Error handling for invalid inputs
* Division by zero protection
